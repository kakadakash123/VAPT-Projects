# Zero Bank - Vulnerability Assessment and Penetration Testing (VAPT)

## Project Overview

Conducted a Vulnerability Assessment and Penetration Testing (VAPT) engagement on the Zero Bank web application to identify security weaknesses and evaluate the application's security posture.

## Scope

- Authentication Testing
- Session Management Testing
- Security Misconfiguration Assessment
- HTTP Security Header Review
- SSL/TLS Security Assessment

## Tools Used

- Burp Suite Professional
- OWASP ZAP
- Nmap
- Wireshark
- Nikto
- SSLScan
- DirBuster

## Methodology

The assessment followed a structured VAPT approach:

1. Reconnaissance
2. Enumeration
3. Vulnerability Identification
4. Validation
5. Risk Assessment
6. Reporting

## Key Findings

- Authentication Security Weaknesses
- Session Management Issues
- Missing Security Headers
- Security Misconfigurations
- SSL/TLS Configuration Observations

## Deliverables

- Security Assessment Report
- Risk Analysis
- Remediation Recommendations
- Supporting Evidence Screenshots
