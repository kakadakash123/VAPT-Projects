# Zero Bank - Vulnerability Assessment and Penetration Testing (VAPT)

## Project Overview

Performed Vulnerability Assessment and Penetration Testing (VAPT) on a banking web application to identify security weaknesses and assess overall application security posture.

## Scope

* Web Application Security Testing
* Authentication Testing
* Session Management Testing
* Security Misconfiguration Assessment
* HTTP Security Header Review
* SSL/TLS Security Assessment

## Tools Used

* Burp Suite Professional
* OWASP ZAP
* Nmap
* Wireshark
* Nikto
* SSLScan
* DirBuster

## Methodology

Testing activities were conducted following OWASP Top 10 principles and standard VAPT methodology including:

1. Reconnaissance
2. Enumeration
3. Vulnerability Identification
4. Validation
5. Risk Assessment
6. Reporting

## Key Outcomes

* Authentication Security Review
* Session Management Analysis
* Security Header Assessment
* SSL/TLS Configuration Review
* Security Reporting with Remediation Recommendations
